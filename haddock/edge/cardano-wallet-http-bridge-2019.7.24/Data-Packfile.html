Description

Decoder for the rust-cardano packfile format.

A pack file is a collection of bytestring blobs.

The reference implementation is in packfile.rs.

decodePackfile :: ByteString -> Either PackfileError [ByteString] Source #

Decode a Cardano version 1 pack file. The blobs are returned as a list. Decoding is not incremental, and all data is stored in memory.
