mkTx :: Hash "Tx" -> [(TxIn, Maybe Coin)] -> [TxOut] -> Tx t Source #

Extract transaction resolved inputs. This slightly breaks the abstraction boundary of DefineTx which doesn't really force any structure on the resolved inputs. However, here in the DB-Layer, supporting arbitrary shape for the inputs be much more complex and require quite a lot of work. So, we kinda force the format here and only here, and leave the rest of the code with an opaque ResolvedTxIn type.
